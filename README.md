# рҹ“Ң Overview

**FinFlow AI** is a next-generation autonomous AI CFO platform that combines **Generative AI**, **Machine Learning**, **Predictive Analytics**, and **Business Intelligence** into one unified financial operating system.

The platform leverages a conversational AI assistant grounded in your real financial data via a RAG pipeline, 6 autonomous AI agents for finance automation, and an ML forecasting engine powered by Prophet + XGBoost ensemble for revenue predictions with 94.7% accuracy.

The system combines:

- рҹӨ– Conversational AI CFO with RAG pipeline

- рҹҢҗ Real-time KPI dashboards & BI analytics
